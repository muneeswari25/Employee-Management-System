# Employee Management System

A frontend-only Employee Management System built with HTML, CSS, and JavaScript.

## Features
- ✅ Add, edit, and delete employees
- ✅ Search and filter employees
- ✅ View employee details
- ✅ Department management
- ✅ Data persistence with LocalStorage
- ✅ Responsive design
- ✅ Export employee data

## Getting Started

1. Open `index.html` in your web browser
2. Start managing employees!

## File Structure
```
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Global styles
├── js/
│   ├── app.js         # Main application logic
│   ├── employees.js   # Employee data management
│   └── ui.js          # UI manipulation functions
└── README.md          # This file
```

## Usage

### Add Employee
1. Fill in the employee form with details
2. Click "Add Employee" button
3. Employee will be added to the table

### Edit Employee
1. Click the "Edit" button on any employee row
2. Update the employee details
3. Click "Update Employee"

### Delete Employee
1. Click the "Delete" button on any employee row
2. Confirm the deletion

### Search
Use the search bar to filter employees by name, email, or ID

## LocalStorage
All employee data is automatically saved to your browser's LocalStorage and persists across sessions.
